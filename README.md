# Busy Program\n\n这是一个在GitHub Actions上定时运行的实验性程序。\n\n## 功能\n\n- 每次运行时扫描当前目录下的所有文件\n- 调用DeepSeek API来生成代码修改\n- 随机创建彩蛋文件\n- 记录所有活动到日志文件\n- **新增：生成随机诗歌文件**\n- **新增：统计文件数量**\n\n## 运行\n\n程序通过GitHub Actions定时触发，无需手动运行。\n\n## 文件说明\n\n- `main.py`: 主程序文件\n- `activity.log`: 程序运行日志\n- `easter_egg.txt`: 随机生成的彩蛋文件\n- `random_poem.txt`: 随机生成的诗歌文件\n- `LICENSE`: GPL v3许可证\n\n## 许可证\n\n本项目采用GNU General Public License v3.0许可证。\n\n---\n\n*这是一个自我修改的程序，每次运行都可能改变自身行为！*\n\n*最新更新：添加了诗歌生成功能和文件统计功能*
